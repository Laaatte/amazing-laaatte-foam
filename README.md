# WebP 변환툴

서버 업로드 없이 브라우저에서 바로 이미지를 WebP로 변환하는 도구입니다.

배포 주소: [webp.laaatte.com](https://webp.laaatte.com)

## 기능

- PNG, JPG, JPEG, GIF, BMP, SVG → WebP 변환
- 압축률 슬라이더 (1~95%)
- 해상도 축소 슬라이더 (원본 대비 %)
- 파일 변환 전후 용량·해상도 비교 표시
- 개별 다운로드 및 전체 다운로드
- 드래그 앤 드롭 / 파일 선택 모두 지원
- 모든 처리는 사용자 기기 안에서만 이루어지며, 어떤 데이터도 서버로 전송하지 않음

## 제한 사항

- Safari / iOS WebKit은 canvas WebP 인코딩을 지원하지 않아 변환 불가
- 파일당 최대 50MB
- 한 변 최대 16,384px / 총 1억 화소 이하

## 기술 스택

빌드 도구나 외부 라이브러리 없이 바닐라 JS · CSS · HTML만으로 구현했습니다.
